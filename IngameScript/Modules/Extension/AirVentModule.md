# Air Vent Module
[< Modules](../Modules.md)

The air vent module allows players to control air vents for practic uses in airlocks and oxygen capture.

## Commands

### pressurize

Set the air vent to pressurize mode.
```ini
vent/pressurize <AirVent|Group>
```

### depressurize

Set the air vent to pressurize mode.
```ini
vent/depressurize <AirVent|Group>
```
