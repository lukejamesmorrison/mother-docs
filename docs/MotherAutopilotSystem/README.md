# Mother Autopilot System (MAPS)

**Warning:** MAPS is compatible with scripts running Mother Core v1.1 or later. If you are using an older version of Mother Core, please update to the latest version to ensure compatibility.

Mother Autopilot System (MAPS) is a powerful and versatile autopilot system designed to expand the capabilities of grids running [Mother OS](../IngameScript/IngameScript.md) and other scripts built with [Mother Core](../Framework/README.md). It provides advanced navigation, automation, and control features.

To use MAPS, simply run it in a program block on your grid. It will automatically detect and integrate with compatible scripts, allowing you to share commands across your programmable blocks.
